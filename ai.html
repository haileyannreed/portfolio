AI Honors Project

8-Queens Problem: Local Search

This project demonstrates the use of various local search algorithms to solve the classic n-Queens problem. The objective is to arrange n queens on an n × n chessboard so that no two queens can attack each other.

Implemented Algorithms:

  • Steepest Ascent Hill Climbing
  • Stochastic Hill Climbing
  • First Choice Hill Climbing
  • Random Restart Hill Climbing

Project Overview:

Each algorithm starts with a randomized board configuration and applies iterative local search techniques to reach a valid solution (or a local minimum). The project showcases how different optimization strategies behave in the context of search problems.
