Quantitative Finance

Stationary & Forecasting with ARIMA

This project demonstrates how to prepare raw financial time series data for modeling by transforming it into a stationary series. Stationarity is a key requirement in quantitative finance, as it ensures consistent statistical properties over time, allowing for more accurate modeling. After preprocessing, the project uses the ARIMA model to forecast future values, showcasing a full pipeline from raw macroeconomic data to predictive analytics.
